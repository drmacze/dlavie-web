Tentang DLavie

DLavie Launcher adalah aplikasi launcher Android untuk komunitas FIFA 16 Mobile modding. Launcher ini menyediakan sistem instalasi game, patch management, community hub, dan account management dalam satu aplikasi.

Misi Kami

Memberikan pengalaman modding FIFA 16 Mobile yang mudah, aman, dan terstruktur bagi seluruh komunitas. Kami berkomitmen untuk terus mengembangkan launcher dengan fitur-fitur modern dan UI yang clean.

Fitur Utama

  • FIFA 16 Mod Support — Install base data + patch system via Shizuku/root
  • Community Hub — Feed, follow/unfollow, posts, comments, badges
  • DLavie Portal — Connect akun launcher ke web DLavie
  • Security & Privacy — End-to-end encryption, RLS, JWT auth, bcrypt hashing
  • Cross-Platform Sync — Connect launcher ke web, login sekali untuk semua

Privacy & Security

Data Anda aman bersama kami. Kami menggunakan Supabase dengan Row-Level Security, JWT authentication, HTTPS, dan EncryptedSharedPreferences. Password di-hash dengan bcrypt. Kami tidak menyimpan GPS, kontak, SMS, atau riwayat browsing.

Privacy Policy

Last updated: July 7, 2026

1. Information We Collect

Account Information

  • Email address (for authentication)
  • Username and display name
  • Avatar image (if uploaded)
  • Bio and country (optional)

Device Information

  • Android version (for compatibility)
  • App version (for telemetry)
  • Device model (for crash reporting, via Firebase Crashlytics)

Usage Data

  • App events: app_open, login, patch_apply, game_launch, logout
  • Community interactions: posts, comments, follows, likes
  • Presence: last_seen_at timestamp

2. How We Use Your Information

  • Provide and maintain the App
  • Authenticate and authorize user actions
  • Display community content (posts, comments, profiles)
  • Send push notifications (announcements, ticket replies)
  • Analyze app usage to improve features
  • Detect and prevent abuse, spam, and fraud

3. Data Storage

Your data is stored in Supabase (PostgreSQL) hosted on secure cloud infrastructure. Authentication tokens are encrypted using AES-256-GCM on your device.

We do NOT store: passwords in plain text (bcrypt hashed), GPS location, contacts, SMS, or browsing history.

4. Data Sharing

We do not sell or rent your data. We may share data with:

  • Firebase (Crashlytics, FCM) — for crash reporting and push notifications
  • GitHub — for release downloads (IP address only, no personal data)

5. Your Rights

  • Access: You can view your data in the Profile screen.
  • Correction: You can edit your profile, bio, and avatar.
  • Deletion: You can request account deletion via the Issues page.
  • Data portability: You can export your posts and comments.

6. Data Retention

  • Account data: Retained until account deletion is requested.
  • Telemetry events: Retained for 90 days, then automatically purged.
  • Crash logs: Retained for 90 days (Firebase Crashlytics default).

7. Security

  • Row-Level Security (RLS) on all database tables
  • JWT authentication with automatic refresh
  • HTTPS for all API communication
  • EncryptedSharedPreferences for token storage

8. Children's Privacy

The App is not directed to children under 13. We do not knowingly collect data from children.

9. Changes to This Policy

We may update this Privacy Policy at any time. Changes will be posted on this page.
